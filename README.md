# OIBSIP Web Development and Designing Internship

## Task 3: Temperature Converter Website

### Objective

The objective of this task was to create a temperature converter application that allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin using JavaScript.

### Steps Performed

1. Designed the user interface using HTML5 and CSS3.
2. Added input fields and unit selection dropdowns.
3. Implemented temperature conversion logic using JavaScript.
4. Added validation for incorrect inputs.
5. Developed responsive styling for different devices.
6. Tested all conversion combinations.

### Tools & Technologies Used

* HTML5
* CSS3
* JavaScript
* Visual Studio Code

### Outcome

Successfully developed a responsive temperature converter web application capable of accurately converting temperatures between Celsius, Fahrenheit, and Kelvin with a user-friendly interface.
